# Source Policy

## Allowed Sources

Initial source types:

- official Q-Net past exams
- EBS past-exam and answer materials
- user-uploaded past-exam materials when the source can be identified

Publicly viewable does not mean redistributable. Do not commit past-exam PDFs, extracted bodies, choices, official answers, user photos, user scores, or private study logs into a public skill repository unless redistribution rights are confirmed.

Do not use commercial commentary books, paid lecture materials, paid solution PDFs, or copyrighted private notes as bundled skill content.

## Question ID

Use stable IDs for existing past-exam questions.

Example:

```text
REA-2025-1-LAW-001
REA-2024-2-BROKER-018
REA-2023-2-TAX-033
```

Each stored question should have:

- `source_exam_id`
- `question_id`
- `year`
- `subject`
- `question_no`
- `official_answer`
- `source_url` or source note
- `source_type`
- `official_text_hash` when canonical text is stored
- `license_scope`
- `immutable: true`

## Immutability

Once stored, do not edit:

- body
- choices
- official answer
- official text hash

If correction is needed, create a new source version and archive the old one.

## Reissue Eligibility

A question can be used for reissue only when:

- it has an existing `question_id`
- it is inside the allowed year/source range for the current study phase
- the source is verified enough for the learner's use
- the body and choices are rendered from stored source text, not generated by the model

## Old-Law Risk

For 공법, 세법, 중개사법, and other amendment-sensitive areas:

- follow the official answer for that year's exam
- mark old-law risk when using older questions
- prefer recent years for active reissue unless the old item tests a stable structure
- do not present old-law content as current law without verification

## Search Failure

If source matching is uncertain:

1. Preserve the user's wrong-answer note.
2. Mark `source_type: unknown`.
3. Set status to `needs_source`.
4. Do not add it to the reissue queue.
5. Ask for a clearer photo, year, subject, chapter, or question number only if needed.
