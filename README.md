# re-license-hacker: 공인중개사 오답노트 관리 + 시간 관리

공인중개사 시험용 **오답노트 + 시간 최적화** Codex 스킬입니다.

Time optimizer for RE license test. `체크인`과 `마무리`만 입력하면 공부 시간을 자동으로 계산하고, 중간에 올린 오답 사진을 오답노트/복습 큐/내일 계획으로 바꿉니다.

## 핵심 기능

1. 오답노트 관리
   - 오답 사진이나 짧은 메모를 `W1`, `W2`처럼 정리합니다.
   - 왜 틀렸는지, 어떤 작은 실력 단위가 약한지 기록합니다.
   - 검증된 기존 기출만 복습 큐에 넣습니다.

2. 시간 관리
   - `체크인`과 `마무리` 시각으로 공부 시간을 계산합니다.
   - 복습 부채가 쌓이면 새 진도를 잠급니다.
   - 남은 시간에 맞춰 내일 공부 순서를 정합니다.

## 맞는 사람

- 기출 오답을 체계적으로 관리하고 싶은 수험생
- 매일 2-4시간 정도 공부할 수 있는 사람
- 시간이 부족해서 오늘 무엇을 버리고 무엇을 할지 정해야 하는 사람
- 복습이 밀렸을 때 어디서부터 갚아야 할지 헷갈리는 사람

맞지 않는 경우:

- AI가 새 문제를 많이 만들어주길 기대하는 경우
- 모든 오답에 긴 해설을 받고 싶은 경우
- 강의 진도 관리가 더 중요한 경우
- 합격 보장을 기대하는 경우

## 사용법

Codex에서 스킬을 부른 뒤, 아래처럼 씁니다.

```text
$re-license-hacker

체크인
```

공부 중에는 틀린 문제 사진이나 짧은 메모만 올립니다.

```text
[오답 사진 업로드]

이 문제는 왜 헷갈렸는지도 봐줘.
```

끝나면 이렇게 말합니다.

```text
마무리
```

그러면 Codex가 현재 시각으로 공부 시간을 계산하고 아래를 정리합니다.

- 오늘 로그
- 오답 라벨
- 틀린 이유와 약한 micro-skill
- 복습 큐
- 신규 학습 가능 여부
- 내일 공부 순서
- 사용자가 물어본 문제의 답변

## 원칙

- 재출제는 검증된 기존 기출만 사용합니다.
- 문제 본문, 보기, 정답은 만들거나 바꾸지 않습니다.
- 출처가 불명확한 문제는 복습 큐에 넣지 않습니다.
- Q-Net을 우선하고, EBS와 식별 가능한 업로드 자료는 `references/source-policy.md` 기준으로만 씁니다.
- 세부 운영 규칙은 `references/` 문서가 소유합니다.

공개 저장소에는 기출 원문, 보기, 정답표, 해설집, 사용자 사진, 점수, 개인 공부 로그를 넣지 마세요.
