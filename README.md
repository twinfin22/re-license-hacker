# re-license-hacker

공인중개사 기출 오답을 복습 큐로 바꾸는 Codex 스킬입니다.

새 문제를 만들지 않습니다. 틀린 기출을 다시 맞히게 만들고, 복습 부채와 내일 공부 순서를 정리합니다.

## 맞는 사람

- 기출과 오답 반복이 급한 수험생
- 매일 2-4시간 정도 공부할 수 있는 사람
- 틀린 문제를 다시 틀리는 패턴이 있는 사람
- 복습이 밀렸을 때 무엇부터 해야 할지 헷갈리는 사람

맞지 않는 경우:

- AI가 새 문제를 많이 만들어주길 기대하는 경우
- 모든 오답에 긴 해설을 받고 싶은 경우
- 강의 진도 관리가 더 중요한 경우
- 합격 보장을 기대하는 경우

## 사용법

Codex에서 스킬을 부른 뒤, 공부 기록과 오답만 남기면 됩니다.

```text
$re-license-hacker

오늘 공부 마무리.
민법 물권변동 기출을 풀었고, 2024년 민법 12번과 2022년 민법 18번을 틀렸습니다.
2024년 12번은 왜 틀렸는지도 설명해주세요.
```

Codex는 보통 아래를 정리합니다.

- 오늘 로그
- 오답 라벨
- 복습 큐
- 신규 학습 가능 여부
- 내일 공부 순서
- 사용자가 물어본 문제의 답변

## 원칙

- 재출제는 검증된 기존 기출만 사용합니다.
- 문제 본문, 보기, 정답은 만들거나 바꾸지 않습니다.
- 출처가 불명확한 문제는 복습 큐에 넣지 않습니다.
- Q-Net을 우선하고, EBS와 식별 가능한 업로드 자료는 `references/source-policy.md` 기준으로만 씁니다.
- 세부 운영 규칙은 `references/` 문서가 소유합니다.

## 문서 구조

- `SKILL.md`: Codex가 읽는 라우터 문서
- `references/`: 운영 규칙, 복습 스케줄, source policy, 과목별 selector
- `docs/`: 제품 설계와 300시간 전략 참고 자료

## 공개 자료 주의

공개 저장소에는 기출 원문, 보기, 정답표, 해설집, 사용자 사진, 점수, 개인 공부 로그를 넣지 마세요.

공식 시험 정보, 법령 개정, 정답 기준은 Q-Net 등 공식 출처로 확인해야 합니다. 이 스킬은 학습 운영을 돕지만 합격이나 점수 상승을 보장하지 않습니다.
