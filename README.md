# re-license-hacker

공인중개사 시험을 준비할 때, 틀린 기출을 다시 틀리지 않게 관리하는 Codex 스킬입니다.

짧게 말하면:

- 하루 공부가 끝나면 공부 시간과 오답만 남깁니다.
- Codex가 복습 큐와 내일 공부 순서를 정리합니다.
- 새 문제를 만들지 않고, 기존 기출 오답을 다시 맞히게 만드는 데 집중합니다.

이 저장소는 문제집이나 강의가 아닙니다.  
AI가 새 문제를 만들어주는 도구도 아닙니다.  
하루 공부가 끝난 뒤 오답을 정리하고, 복습할 순서와 내일 공부 계획을 잡아주는 운영 방식입니다.

## 누구에게 맞나요?

- 공인중개사 시험을 준비하는 수험생
- 강의 완강보다 기출과 오답 반복이 더 급한 사람
- 공부 시간은 많지 않지만 매일 2-4시간은 확보할 수 있는 사람
- 틀린 문제를 다시 틀리는 일이 반복되는 사람
- 복습할 게 쌓여도 어디서부터 갚아야 할지 헷갈리는 사람

맞지 않는 경우도 있습니다.

- AI가 새 문제를 많이 만들어주길 기대하는 경우
- 모든 문제에 긴 해설을 받고 싶은 경우
- 기출보다 강의 진도 관리가 더 중요한 경우
- 법령 개정이나 공식 정답 확인 없이 자동으로 맡기고 싶은 경우
- 합격을 보장해주는 프로그램을 기대하는 경우

## 이 스킬이 하는 일

하루 공부 후 아래만 Codex에 알려줍니다.

- 공부 시작 시간
- 공부 종료 시간
- 오늘 푼 과목/단원
- 틀린 문제 번호 또는 사진
- 꼭 물어보고 싶은 질문

그러면 Codex가 아래를 정리합니다.

- 오늘 공부 로그
- 오답 번호: `W1`, `W2`, `W3`처럼 라벨링
- 왜 틀렸는지
- 어떤 작은 실력 단위가 약한지
- 다시 풀어야 할 기존 기출
- 복습 부채가 얼마나 쌓였는지
- 내일 새 진도를 나가도 되는지
- 내일 공부 순서

핵심은 하나입니다.

> 많이 공부한 척하지 말고, 이미 틀린 기출을 실전에서 맞히게 만들기.

## 중요한 원칙

이 스킬은 새 문제를 만들지 않습니다.

허용:

- 기존 기출 문제 번호로 다시 풀 문제 고르기
- 틀린 이유 정리
- 복습 큐 만들기
- 내일 공부 계획 만들기
- 질문한 문제만 짧게 해설하기

금지:

- AI가 새 지문 만들기
- AI가 새 보기 만들기
- 기존 기출의 숫자, 기간, 주체를 바꿔 변형 문제 만들기
- 출처가 불명확한 문제를 재출제 큐에 넣기
- 모든 오답에 자동으로 긴 해설 쓰기

## 사용 예시

Codex에서 이렇게 말하면 됩니다.

```text
$re-license-hacker

오늘 공부 시작 08:40, 종료 12:05.
민법 물권변동 단원별 기출을 풀었습니다.
오답은 W1 2024년 민법 12번, W2 2022년 민법 18번입니다.
W1은 왜 틀렸는지 설명도 받고 싶습니다.

오늘 로그, 복습 큐, 내일 계획을 만들어주세요.
```

Codex는 이런 형태로 답합니다.

```text
오늘 판정:
- 복습 부채:
- 신규 학습 가능 여부:
- 가장 위험한 과목:

내일 계획:
1.
2.
3.

복습 큐:
- due today:
- overdue:
- rebuild:

오답 재출제:
- W1:
- W2:

질문 답변:
- Q1:
```

```text

```

직접 설치할 줄 아는 경우에는 이 저장소를 Codex의 skills 폴더에 넣으면 됩니다.

```bash

```

설치 후 새 Codex 대화에서 `$re-license-hacker`라고 부르면 됩니다.

## 목표

- 공시법/세법은 깊게 파기보다 맞힐 것만 반복합니다.
